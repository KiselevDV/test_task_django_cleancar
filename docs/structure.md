# Структура проекта

```
.
├── config/             # Основные настройки Django
├── docker/             # Файлы Docker
├── docs/               # Документация
├── trading/            # Логика работы с торговыми данными
├── manage.py           # Управление Django
├── poetry.lock         # Фиксация зависимостей
├── pyproject.toml      # Зависимости и настройки проекта
├── README.md           # Основной файл документации
└── .env                # Переменные окружения
```
